Constellation

DEMO LINK

This web application is designed to allow users to easily find businesses near them, as well as inform them of the quality of the location. In addition, this application will suggest alternative locations for the same business that have better ratings, as well as different businesses of higher quality. This will be done by accessing the APIs for Yelp and Google and displaying combined ratings and reviews. We will design a novel ranking algorithm, combining information from Google and Yelp.

Requires:

  • Yarn https://yarnpkg.com/lang/en/docs/install/
  • Nodemon "npm install -g nodemon" https://www.npmjs.com/package/nodemon

To run node.js and react:

  • > git pull
  • > cd client
  • > yarn install
  • > cd ..
  • > yarn install
  • > yarn dev

Data

We used the Yelp and Google Places APIs to retrieve information and reviews about businesses.

  • Yelp API - https://www.yelp.com/developers/documentation/v3
  • Google API - https://developers.google.com/places/

Additionally, Yelp has an offline dataset (https://www.yelp.com/dataset/challenge) that we used to extract keywords.
